What is RxJS?
  • RxJs Stands for Reactive eXtensions for JavaScript
  • RxJS helps in dealing with ASync Data Stream
  • Without RxJS, Dealing with Async Data Stream would have been Tedious.
    As Data Streams will emit data at certain interval of time. And each time this Data is emitted, you need to handle this emitted data.
    Also, the time at which the data will emit is not known. So it would be very difficult to code such data
  • The Async Data Streams are usually known as Observable and the Listener who is listeining to these data streams is known as Observer
  • Observable could be anything -> A Function,an Object, a HTTP Response, etc
  • Observer (the Listener) could be anything -> A Function, an Object , etc
