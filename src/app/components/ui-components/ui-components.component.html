Directive
Directives in Angular is used to manipulate DOM Elements.
There are 3 types of Directives in Angular
  • Structural Directive: These kind of directives are use to manupulate the DOM by displaying or hiding elements based on a truthy or falsy value =>Conditions
    Some Examples: *ngFor, *ngIf ,*ngSwitch
  • Attribute Directives: This is use to change appearance of the element in the DOM. Some examples: [ngStyle], [ngClass]
  • Component Directives: The directives based on components are known as component directives. Each component has @Input and @Output properties. Whenever you try to bind any such properties, you are making use of component directives.
